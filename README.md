### Data Engineering for Data Scientists

Materials for the Live Online Training



#### Install

```sh
pip install sklearn-pandas
pip install fire
pip install rollbar
pip install python-dotenv
pip install apache-airflow
pip jupyter
```

#### Run Jupyter Notebook

```sh
$ jupyter-notebook --ip=<fqdn>
```
